# 🔗 Lab 2: Semantic Similarity with Embeddings

## Objective

Learn how to measure **semantic similarity** between words and sentences using vector embeddings and cosine similarity.

## Key Concepts

- Generate embeddings using Gemini API
- Implement cosine similarity from scratch
- Compare word-to-word and sentence-to-sentence similarity
- Build a mini **semantic search engine**
- Understand how machines compare meanings, not just surface-level words

## What You'll Build

- A similarity function using cosine distance
- A comparison tool for any two texts
- A small semantic search engine over multiple sentences

## Key Finding

> Cosine similarity on Gemini embeddings accurately captures semantic relationships — synonyms score high, unrelated words score near zero. This is the foundation for semantic search systems.
